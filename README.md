# 📌 TaskManager

Sistema de gerenciamento de tarefas em Java com interface via terminal.

## 🚀 Funcionalidades
- 📝 Criar novas tarefas
- 📋 Listar todas as tarefas
- ✅ Marcar tarefas como concluídas
- 🗑️ Remover tarefas existentes
- ⭐ Suporte a tarefas prioritárias

## 🛠️ Tecnologias
- Java 17
- NetBeans
